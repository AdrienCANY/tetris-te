Chemin de l'executable : build/tetriste.exe

Commandes:
Q -> déplacement vers la gauche
D -> déplacement vers la droite
E -> rotation sens horaire
A -> rotation sens anti-horaire
Z -> Mise en réserve
( Note : Il faut avoir posé une pièce pour pouvoir le réutiliser )
S -> Descente rapide
Espace -> descente instantanée
Echap -> Pause
G -> Activer / Désactiver l'affichage de la grille
